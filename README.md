The Student Grievance Management System is a full-stack MERN application developed to provide a streamlined platform for students to register, log in, and manage their grievances related to academics, hostel, transport, or other issues. The system uses React for the frontend, Node.js and Express for the backend, and MongoDB Atlas for database management, with JWT-based authentication and bcrypt for secure password handling. Students can submit, view, search, update, and delete grievances, while protected routes ensure that only authenticated users can access their data. The application is deployed with the frontend on Vercel and the backend on Render, making it accessible online and demonstrating real-world full-stack development and deployment practices.
